# Download SAM 2.1 Checkpoints

To proceed with **SAM 2.1 fine-tuning**, you must first download the pre-trained model weights (checkpoints) and place them in this directory.

### Official Repository
All official checkpoints and detailed model information are maintained by Meta (Facebook Research) at the following repository:
* **URL:** [https://github.com/facebookresearch/sam2.git](https://github.com/facebookresearch/sam2.git)

### How to Download
Please refer to the **Official README** in the link above. The Meta team provides direct download links for various model sizes (Tiny, Small, Base, and Large) as well as automated download scripts.

> **Note:** Ensure that the downloaded `.pt` files are located in the correct folder (e.g., `./checkpoints/`) before starting the training process.
